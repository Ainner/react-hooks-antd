## 安装依赖

```bash
yarn
```

## 启动本地开发服务

开发依赖环境 Node 14+，默认端口 8080。

```bash
yarn start
```

## 打包编译

默认输出 dist 文件夹

```bash
yarn build
```

## 检查代码格式

配置文件 `.eslintrc` 参考官网 https://eslint.org/

```bash
yarn lint
```

## 格式化代码

配置文件 `.perttierrc` 参考官网 https://prettier.io/

```bash
yarn prettier
```

## 备注

> public 目录内的资源都会按照原路径 Copy 至 dist 目录。
